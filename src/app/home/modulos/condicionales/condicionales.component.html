🔀 Estructuras Condicionales

🧠 ¿Qué es una estructura condicional?

En programación, una estructura condicional es una instrucción que permite que el programa tome decisiones dependiendo de si una condición es verdadera o falsa.

Es como si el programa se encontrara con una pregunta y, según la respuesta, siguiera un camino u otro.

Ejemplo:

Si tienes hambre → comes.

Si no tienes hambre → no comes.

🔧 Elementos fundamentales de una estructura condicional

• Condición: Es una comparación o evaluación que puede dar como resultado verdadero o falso. Ejemplo: edad >= 18 (¿Es la edad mayor o igual a 18?)
• Bloque verdadero: Son las instrucciones que se ejecutan si la condición se cumple (es verdadera).
• Bloque falso: Son las instrucciones que se ejecutan si la condición no se cumple (es falsa).
• Camino alternativo: Algunas estructuras condicionales permiten evaluar más de una condición con else if o condicionales anidados.

🎓 ¿Por qué se llaman "condicionales"?

Porque la ejecución del código depende de una condición que se debe cumplir. Si la condición es verdadera, el programa hace una cosa. Si es falsa, hace otra o no hace nada.

📍 Ejemplo en la vida real

Si en el semáforo hay luz verde → avanzas.

Si no hay luz verde (amarilla o roja) → te detienes o esperas.

📦 Metáfora: Camino con bifurcaciones

Imagina que vas caminando y llegas a una bifurcación con dos letreros:

Uno dice "Si está soleado → sigue por aquí hacia el parque".

Otro dice "Si está lloviendo → sigue por aquí hacia el café".

En ese punto, observas el clima (condición) y tomas el camino según el resultado. El programa hace exactamente lo mismo: evalúa la condición y ejecuta la ruta de instrucciones correspondiente.

🖥 ¿Dónde se usan los condicionales?

Los condicionales están en casi todo tipo de programa:

• Para verificar permisos de acceso.
• Para decidir si un usuario ve cierta información.
• Para ejecutar una acción solo si hay datos disponibles.
• Para determinar qué mensaje mostrar dependiendo de una respuesta.
• Para controlar el flujo de un videojuego (ganar, perder, seguir).

🧠 Importancia de las estructuras condicionales

Las estructuras condicionales son esenciales porque:

• Permiten adaptar el comportamiento del programa a diferentes situaciones.
• Son la base para tomar decisiones complejas.
• Combinadas con variables, bucles y funciones, hacen que el programa sea flexible e interactivo.

🧪 Curiosidades

Algunos lenguajes permiten condicionales en una sola línea, como en Python:

mensaje = "Mayor de edad" if edad >= 18 else "Menor de edad"
